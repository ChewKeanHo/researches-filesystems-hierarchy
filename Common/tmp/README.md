# `tmp`

[![banner](/.internals/trademarks/animated-banner_1200x100.svg)](#)

This directory houses all temporary production files.

While the directory is temporary, interrupting any file while the
processing is on-going can cause unwanted destructions and interruptions.




## Temporary Natures

[![banner](/.internals/trademarks/animated-banner_1200x100.svg)](#)

Files housed in this directory **MUST NOT** be permanently dependent.
They must be temporary in nature and can be deleted at anytime outside
processing hours.
