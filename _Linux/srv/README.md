# `/srv`

[![banner](/.internals/trademarks/animated-banner_1200x100.svg)](#)

This is the directory housing site-specific data served by Red Hat Linux. This
directory gives users the location of data files for a particular service, such
as FTP, WWW, or CVS.

This directory is **ENTIRELY OPTIONAL** depending on the type of Linux-based
OSes. By default, it is absent. Only Ret Hat Linux is using this directory.

You **SHOULD NOT** place anything here.
