# `/mnt`

[![banner](/.internals/trademarks/animated-banner_1200x100.svg)](#)

This is the base directory housing temporary mounted removable media
device such as but not limited to CD, floppy disks, USB storage devices,
external disks, decrypted partitions, or volume managed partitions.

This is used by sysadmins and root user to debug a storage device or
partition.

For permanent or production use mounting, use `/media` base directory
instead.
